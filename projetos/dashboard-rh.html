Dashboard RH

Projeto de PowerBI e Excel, desenvolvido para auxiliar na análise de dados de contratação de funcionários para o RH

Funcionalidades

A empresa fictícia necessita de um dashboard para facilitar a visualização das contratação dos funcionários ao longo dos anos.

  • Cards: Para uma rápida visualização de indicadores importantes como: Respostas da Pesquisa de Satisfação, Cargos ocupados pelos novos contratados e a quantidade de novos funcionários.
  • Filtros: Para segmentar os dados referentes a um período específico.
  • Gráficos: Para visualizar rápidamente a quantidade de contratação baseada onde a busca da vaga foi feita, quantidade de contratações por ano e a satifação dos clientes com as métricas consideradas importantes: carga horária, salário e saúde.
